## Features

## Principles

- Offline first
- Composable, leave out what you do not need
- Zero configuration
- Zero option flags
- Composable, just like lego bricks you can you make your own player, review or builder
- Serverless, rendering and scoring can be done on the frontend
- Responsive, support for different screen resolutions
- Extensible with customized qti-interactions ( see qti-components-custom )
- No clever code (ie hacks!)
- Embraces different platforms ( touch/mouse ) and their way of input
- Pluggable responseprocessing ( see qti-components-responseprocessing )
- Framework agnostic, use with Vue, React, Angular or plain vanilla
- Lightweight, core bundle is just 17kb not minified
- Few dependencies, only one dependency on @Lit
- Stateless, event based, no state is managed
- Expose relevant components to light dom for easy styling
- Examples in plain vanilla javascript
